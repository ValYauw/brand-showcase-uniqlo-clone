# Uniqlo Brand Showcase Clone

Brand showcase web and mobile application (clone of the Uniqlo website and mobile app).

## Tech Stack 

 - Vite (development platform)
 - React
 - React Native
 - MUI
 - GraphQL Service
 - Node.js (Express) Microservices with PostgreSQL Database
